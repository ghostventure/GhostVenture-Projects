Support.

Use this page when install, setup, account checks, or trading readiness need a clean pass.

Install check

Confirm the `.deb` checksum, install package, launch Timmy, and check that the Setup tab opens.

Broker check

Verify the broker credentials inside Timmy before saving. If verification fails, Timmy should not save the profile.

Trading check

Start in paper mode, review watchlists, inspect buying power, then move to guarded live only when the desk is ready.
