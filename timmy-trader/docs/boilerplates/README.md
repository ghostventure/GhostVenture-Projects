# Timmy Setup Boilerplates

These are copy-safe examples for common Timmy setups. They are not active config
files, and they do not contain real broker credentials.

Use the native `Setup` tab for real Webull keys, account checks, and saving the
local profile. Save still stays locked until verification passes.

## Files

- `paper-desk.env.example`: paper-first setup for testing the market desk.
- `live-guarded.env.example`: live account setup shape with guarded defaults.
- `runtime-reliability.env.example`: watchdog, safe mode, and health settings.
